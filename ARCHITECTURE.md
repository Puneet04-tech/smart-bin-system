## Search Algorithm

```
Input: User location, all bins, filters

Algorithm:
1. Filter bins by product type (if applicable)
2. For each bin:
   - Calculate distance using Haversine formula
   - Check if within search radius (50km)
3. Create list of nearby bins
4. If nearby bins exist:
   - Sort by distance (ascending)
   - Apply search query filter
   - Apply status filter
   - Return filtered & sorted results
5. If no nearby bins:
   - Remove radius constraint
   - Calculate distance for all bins
   - Sort by distance (ascending)
   - Return 10 closest nationwide bins

Time Complexity: O(n log n) for sorting
Space Complexity: O(n) for distance calculations
```
